# NeuroTrust-EEG

Explainable and efficient brain-inspired AI for low-data EEG classification.

NeuroTrust-EEG is a research-oriented project focused on trustworthy AI for EEG time-series classification. The project combines compact neural models, subject-independent evaluation, uncertainty calibration, explainability methods and brain-inspired prototype memory.

> This is a research prototype. It is not a medical device and must not be used for diagnosis.

## Research question

Can a compact, explainable and brain-inspired model classify EEG windows under low-data and subject-independent conditions while remaining calibrated and computationally efficient?

## Core goals

- Build a reproducible EEG classification pipeline.
- Avoid data leakage using subject-independent splits.
- Evaluate performance under low-data conditions.
- Add uncertainty calibration and explainability.
- Explore prototype-based memory as a brain-inspired component.
